🎵 오뮤직 솔루션 가이드

아래 단계를 순서대로 완료하시면 솔루션 사용 준비가 완료됩니다!

0 / 7 단계 완료
1

💰 강사 급여 등록

수업 유형별 단가와 기타 수당을 설정합니다

▼
강사관리 → 급여 버튼 클릭
💡 팁: 급여 설정은 나중에 언제든 수정할 수 있습니다. 하지만 과거 데이터에는 영향을 주지 않으니 처음에 정확하게 입력해주세요.
2

🎫 수강권 등록

결제 시 사용할 수강권을 미리 등록합니다

▼
당일 결제 → 수강권 추가/수정 버튼 클릭
✅ 예시: 수강권 이름: "오픈이벤트 3개월" / 가격: 480,000원 / 회차: 12회
3

👨‍🏫 강사 등록

강사님 정보와 로그인 계정을 생성합니다

▼
강사관리 → 강사 추가 버튼 클릭
⚠️ 주의: 비밀번호는 강사님이 로그인 후 직접 변경하도록 안내해주세요.
4

📋 강사별 수강생 업로드

엑셀 파일로 수강생 정보를 일괄 등록합니다

▼
강사관리 → 강사의 수강생 수 클릭 → 엑셀 업로드 버튼

📥 엑셀 템플릿 다운로드

📄 수강생 등록 템플릿 다운로드

🗑️ 수강생 삭제 방법

방법 1 - 완전 삭제:
강사관리 → 강사별 수강생 선택 → 선택 삭제 / 전체 삭제
※ 데이터가 완전히 삭제됩니다
방법 2 - 소프트 삭제 (권장):
당일 결제 → 수강생 상세 → "학생삭제" 버튼 클릭
※ 데이터가 보존되어 본사에서 확인 가능합니다
