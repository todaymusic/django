SECTION 3

출석/결석 처리 (오수완)

수업 완료 후 출석 또는 당일결석 처리를 진행합니다. "오수완"은 "오늘 수업 완료"의 줄임말이에요!

1

수업 칸 클릭

주간 스케줄에서 완료할 수업(흰색 배경)을 클릭합니다.

📝 수업 처리 팝업
📝 수업 처리 (김민준)
👍 오수완 🚫 결석
🗑️ 취소
2

피드백 작성

오늘 수업 내용, 진도, 특이사항 등을 간단히 기록합니다.

3

버튼 선택

• 👍 오수완 : 정상 출석 처리 (잔여 횟수 1회 차감)
• 🚫 결석 : 당일 결석 처리 (잔여 횟수 1회 차감)
• 🗑️ 취소 : 수업 자체를 삭제 (횟수 차감 없음)

⚠️ 이미 출석 처리된 수업(초록색)을 클릭하면 "이미 출석된 수업입니다" 알림이 뜹니다. 수정이 필요하면 원장님께 문의하세요!
